一线销售的真实困境

200+ 在手客户，三大痛点让转化率持续下滑

信息分散、缺乏智能辅助、重复性操作繁重——这三个问题不解决，客户体验与成交率就无法真正提升。

痛点 01
客户信息分散，难以整合
客户资料横跨 CRM、微信、试驾系统、金融系统等多个平台。销售专家需要在不同工具间反复切换，耗费大量精力仍无法形成完整的客户视图，接待前无法做到真正了解客户。
「我记不住每位客户上次关心什么，每次接触都像从头开始。每天光找信息就要花 1-2 小时。」
→ 导致：接待准备不足，客户感受差，复购转介绍率低
痛点 02
缺乏智能辅助，应对被动
面对客户的竞品质疑、价格追问或个性化需求，销售专家只能凭经验和记忆应对。知识储备参差不齐，新人难以快速补位，错过成交时机。竞品更新迭代快，信息滞后是常态。
「客户问我们和问界、蔚来的区别，我脑子里一片空白，只能先说'我查一下'，客户当场就走了。」
→ 导致：竞品应对失位，成交关键时机流失
痛点 03
重复操作繁重，精力被消耗
撰写跟进内容、整理客户记录、制作报价对比……大量时间被这些低价值、可自动化的操作占据。真正用于深度客户沟通与临门一脚的有效时间严重不足。
「一天里真正和客户有效沟通的时间，可能不到两小时。剩下的时间都在填表格、想发什么。」
→ 导致：高价值客户跟进不及时，转化率持续下滑
新人培养周期长
车型知识复杂、销售流程繁琐，新人通常需要 2-3 个月才能独立接待，期间大量机会白白流失。
经验无法沉淀复用
老销售的宝贵经验和成熟话术只存在于个人脑中，无法结构化传递给团队，造成整体能力参差不齐。

工作流程

四步上手，即刻生效

从自然语言输入到可直接使用的建议，Sales Agent 将复杂的 AI 能力包装为极简的操作体验。

01
描述销售场景
用自然语言说出当下情况——客户在问什么、处于哪个阶段、有哪些顾虑。不需要任何格式要求，就像和同事说话一样自然。
02
Agent 即时分析
对应 Agent 结合客户历史、车型知识与销售策略，在 2 秒内生成有针对性的建议与话术，自动匹配最适合的 Skill 组合。
03
采用或精细调整
直接使用推荐内容，或告诉 AI 你想要的方向——「更简洁一些」「强调家庭场景」，持续迭代直到满意为止。
04
积累 · 持续进化
每次对话都沉淀为 Agent 的长期记忆，用得越多越懂你的客户与风格，形成真正属于你的专属知识库。

常见问题

你可能想了解的更多细节

Sales Agent 和普通 AI 助手有什么区别？
Sales Agent 专为理想汽车销售场景深度训练，内置全系车型知识、竞品数据和销售话术体系，而不是通用 AI。它能理解销售上下文，知道你在什么阶段、面对什么客户，给出真正可以直接使用的建议。
使用 Sales Agent 需要专门培训吗？
不需要。Sales Agent 采用自然语言交互，就像和同事聊天一样描述你的场景即可。大多数销售专家在首次使用后 15 分钟内即可上手，完整功能探索通常在 1-2 天内完成。
客户信息的数据安全如何保障？
Sales Agent 采用企业级数据加密存储，所有客户数据在理想汽车私有云环境中处理，不会上传至任何第三方平台。访问权限根据角色严格控制，符合企业数据合规要求。
支持哪些设备？只有 iPad 吗？
当前版本专为 iPad 优化，充分利用大屏幕的展示优势，适合在展厅内使用。iPhone 版本正在开发中，将支持外出时的轻量级使用场景，预计近期推出。
车型知识库多久更新一次？
车型参数、配置信息和活动政策由总部实时同步，通常在官方发布后 24 小时内完成更新。竞品信息每周进行一次系统性更新，确保对比数据的时效性。
如何申请使用资格？
当前处于内测阶段，通过展厅店长统一申请或直接填写下方表单提交使用申请。我们会优先向重点展厅和有明确需求的团队开放，预计在申请后 1-2 周内收到回复。
